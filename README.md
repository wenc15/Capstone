# Capstone: Growin

## Testing the Latest Pre-release Build

Developers can download the latest Windows test build from the repository's **Releases** section.

Steps:
1. Open **Releases**
2. Download the newest pre-release `.exe`
3. Run the executable
4. If Windows shows a warning, click **More info** → **Run anyway**

Please verify:
- app launch
- backend auto-start
- focus session behavior
- whitelist behavior
- tracking features
