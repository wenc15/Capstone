# GrowIn Backend (src)

This folder contains the ASP.NET Core backend for GrowIn.

## Overview

The backend handles the core logic and local data processing of GrowIn, including:

- focus session management
- whitelist checking
- website and application usage tracking
- statistics and achievement logic
- credit and reward handling
- food and skin gacha logic
- pet-related features

## Tech Used

- ASP.NET Core
- C#
- SQLite
- REST API
- WebSocket

## Requirements

Please install the following before running the backend locally:

- .NET SDK 8.0 or a compatible version

You can check your installation with:

bash
dotnet --version


## Run the Backend Locally

Open a terminal in the src folder and run:

- dotnet run

If the backend starts successfully, you should see output similar to:

Now listening on: http://localhost:5024
